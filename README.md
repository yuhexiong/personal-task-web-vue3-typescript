# Personal Task

A server to record all tasks.  
Two pages, home and task, with preset fake task data and the data is saved in localStorage.  

## Overview

- Language: TypeScript v5.0.3
- Web FrameWork: Vue3 v3.3.4


## Run

### install dependencies

```bash

npm install

```


### build

```bash

npm run build

```

### run

```bash

npm run dev

```
